# Installation

## Requirements

- Go 1.18 or later

## Install

```bash
go get github.com/verse91/fuzzyvn
```

## Dependencies

FuzzyVN uses the following dependencies:

| Package | Purpose |
|---------|---------|
| `github.com/sahilm/fuzzy` | Fuzzy string matching |
| `golang.org/x/text` | Unicode normalization |

## Import

```go
import "github.com/verse91/fuzzyvn"
```
