# Cài đặt

## Yêu cầu

- Go 1.21 trở lên

## Cài đặt

```bash
go get github.com/verse91/fuzzyvn
```

## Thư viện phụ thuộc

| Package | Mục đích |
|---------|----------|
| `github.com/sahilm/fuzzy` | Tìm kiếm chuỗi mờ |
| `golang.org/x/text` | Chuẩn hóa Unicode |

## Import

```go
import "github.com/verse91/fuzzyvn"
```
